# Software Access Request

Last updated: 2026-06-28

Request approved business software through the helpdesk. Include the application name, work purpose, manager name, and access level required. The application owner may need to approve the request before support assigns a license or role.

Do not use a colleague’s account or request shared credentials. If you already have access but receive an error, include the exact error text or screenshot. For urgent access, state the deadline and business impact. Support will confirm when access is granted; some applications require signing out and back in before new permissions appear.
