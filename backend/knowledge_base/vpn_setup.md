# VPN Setup Guide

Last updated: 2026-07-15

Use the company VPN whenever you need internal services while working outside the office. Install the approved VPN client from the software portal, then sign in with your company email and password. Select the company profile before connecting. If multi-factor authentication is enabled, approve the prompt in your authenticator app.

If connection fails, confirm that your internet works by opening a public website. Disconnect and reconnect the VPN client, then restart the client if necessary. Capture the exact error message and time before raising a ticket so support can investigate account, profile, or network issues.
