# Finance Drive Access

Last updated: 2026-04-18

The Finance drive contains restricted financial reports and is available only to approved employees. To request access, submit a ticket with the folder path, business purpose, manager name, and whether you need read-only or edit access. Finance management must approve the request before support updates permissions.

If previous access stopped working, provide the folder path and access error. Do not move confidential files to personal storage or share folders with unauthorized users. Permission changes are normally applied during business hours. Once access is granted, close and reopen File Explorer or sign out and back in if the folder is not immediately visible.
