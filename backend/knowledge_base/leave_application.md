# Leave Application Process

Last updated: 2026-05-20

Apply for planned leave through the employee leave portal before the requested dates. Select the correct leave type, enter start and end dates, and add a note if required. Your manager receives the request for approval and the leave balance is updated after approval according to company policy.

For sick leave or an urgent absence, notify your manager as soon as possible and submit the portal request when able. If the portal shows an incorrect balance, include leave type, dates, and a screenshot in an HR ticket. Do not submit duplicate requests to work around a portal error.
