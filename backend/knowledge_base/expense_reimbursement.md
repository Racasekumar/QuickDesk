# Expense Reimbursement

Last updated: 2026-03-22

Submit reimbursement requests through the approved expense system within the company’s submission period. Attach clear receipts, select the correct category, and provide the project or cost center when required. Explain why the expense was necessary for company work.

Before submitting, confirm that amount, currency, date, and merchant match the receipt. Personal expenses and unsupported items cannot be reimbursed. If a report is rejected, review the comment and correct the requested information. For system access problems, create a ticket with the error and screenshot if available; do not attach payment-card details.
